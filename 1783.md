# 1783: 首尾相连
## Description

给定一个长度小于等于 $400,000$ 的字符串，在每个字符串中求所有既是前缀又是后缀的子串长度。
例如：ababcababababcabab，既是前缀又是后缀的有：ab、abab、ababcabab、ababcababababcabab。

## Input format

若干行，每行一个字符串，字符串中没有空格。

## Output format

对于每一个字符串，输出一行，这个字符串所有既是前缀又是后缀的子串的长度，按照从小到大的次序，数字之间空格分开。

## Samples

Input #1
```
ababcababababcabab
aaaaa
```
Output #1
```
2 4 9 18
1 2 3 4 5
```
